Создание документов с использованием HTML

Существует множество способов для создания документов, используя язык разметки гипертекста HTML. Все они отличаются друг от друга лишь методами добавления тегов на страницу и редактированием её содержимого. Результат же, как правило, всегда выглядит примерно одинаково. Мы будем использовать классический способ для вёрстки - разметка гипертекста с помощью редакторов кода.

План создания простого документа

      1. Запустить любой удобный для Вас редактор кода;
      2. Написать следующий код:
      <HTML>
      <HEAD>
      </HEAD>
      <BODY>
      </BODY>
      </HTML>
      3. Наполнить блоки HEAD и BODY содержимым;
      4. Сохранить документ под именем index.html;
      5. Открыть получившийся файл с помощью браузера.
